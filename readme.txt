Sell directly to verified, licensed beauty professionals.
 
== Description ==

Licensify is a groundbreaking WooCommerce plugin that revamps the online commerce checkout experience, leveraging patented automated license verification technology. Its primary objective is to preserve the sanctity of the professional beauty industry.

**Backend Technicalities - How It Functions?**

*INTEGRATED API TECHNOLOGYY*

* License verification is accomplished through an overlaid window pop-up, ensuring that the professional user remains on the partner site

* Utilizes a powerful and dynamic API that swiftly returns the result of the verification request

* Customizable login button that can be strategically positioned anywhere on the website for user convenience

*COMPREHENSIVE DATA STORAGE SYSTEM*

Automated synchronization with State Barbering and Cosmetology Boards for real-time updates on licensing status and activities.

*FULLY MANAGED SOLUTION*

* Comprehensive quality control across all 50 states, ensuring license verification within every state

* Provides 1:1 technical support, featuring automatic alerts for verification errors or time-outs

*DATA PRIVACY AND SECURITY*

* Ensures complete protection of customer and sales data

* Customer identifying data and credit card information are neither stored nor captured

* No exchange or sale of brand sales or customer information to third parties 

* Data processing is conducted in compliance with statutory permissions and user consents

**Plugin Features**

Licensify is committed to safeguarding the exclusivity of professional-only beauty products, allowing their sale solely to verified beauty professionals.

The plugin is versatile, integrating seamlessly with two kinds of e-commerce models: those that require account protection and those that permit open browsing. It is an ideal solution for onboarding new professionals and eradicating the need for manual verifications.

Products can be individually marked as 'beauty pro only' or the entire store can be configured to grant access only post successful verification.

== Installation ==

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended

To do an automatic installation of the Licensify plugin, log in to your WordPress site dashboard, next to the Plugins menu. Then click "Add New" button and in the search field type "licensify" then click "Search Plugins". Click "Install Now" to install the plugin.
